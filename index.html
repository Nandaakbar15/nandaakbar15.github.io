About Me

Halo! Saya Alvinda Akbar, seorang Fullstack Web Developer dengan pengalaman membangun backend yang efisien dan frontend yang interaktif. Saya memiliki pengalaman sebagai Backend Engineer di PT. Supernova Palapa Nusantara selama 3 bulan dan sebagai Freelance Fullstack Developer untuk membangun website perusahaan CV. Gilang Kencana Mulya.

Saya menguasai Laravel, PHP, MySQL, dan Node.js (ExpressJS, NestJS) untuk pengembangan backend, serta ReactJS, JavaScript, dan Bootstrap untuk frontend. Saya juga memiliki pengalaman dalam membangun dan mengelola REST API, autentikasi pengguna (Sanctum, JWT), serta optimasi database.

Selama magang, saya berkontribusi dalam mengembangkan fitur backend menggunakan NestJS, mengoptimalkan database dengan MongoDB, serta berkolaborasi dengan tim frontend untuk integrasi API.

Sebagai Freelance Developer di CV. Gilang Kencana Mulya, saya mengembangkan website perusahaan berbasis Laravel, mencakup pengelolaan data kontraktor, personil, dan proyek. Saya juga menerapkan sistem autentikasi, manajemen database, dan deployment ke server.

Saya memiliki minat dalam membangun website yang scalable, modern, dan inovatif, serta siap berkontribusi dalam proyek yang menantang di industri teknologi.
